## Installation

To install follow these steps:

1. Clone the repo
   ```sh
   git clone https://github.com/tobicodigo/vue_todoApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Open in Terminal and run
   ```sh
   npm run dev
   ```
